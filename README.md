# Calculadora JavaScript

Calculadora simple creada como proyecto educativo para practicar desarrollo web con HTML, CSS y JavaScript.

## Descripción

Este proyecto contiene una calculadora funcional con operaciones básicas. Incluye una pantalla para mostrar los números y resultados, botones para dígitos y operadores, y un diseño moderno con CSS Grid.

## Características

- Operaciones básicas: suma, resta, multiplicación y división
- Pantalla para mostrar expresiones y resultados
- Botones interactivos con efectos hover
- Diseño responsive con CSS Grid
- Uso de JavaScript para la lógica de cálculo

## Estructura del proyecto

- `index.html` — página principal de la calculadora
- `script.js` — lógica de JavaScript para las operaciones
- `style.css` — estilos para el diseño de la calculadora
- `README.md` — documentación del proyecto

## Cómo usar

1. Clona o descarga el repositorio.
2. Abre `index.html` en un navegador web.
3. Usa los botones para ingresar números y operadores, y presiona "=" para calcular el resultado.

## Recomendaciones

- Asegúrate de que JavaScript esté habilitado en el navegador.
- Puedes personalizar los colores, tamaños de fuente y agregar más funciones como raíz cuadrada o porcentajes.
- Para ver el diseño responsive, reduce el ancho del navegador o prueba en un dispositivo móvil.

## Tecnologías

- HTML
- CSS
- JavaScript

## Notas

Este proyecto es ideal para aprender las bases de JavaScript y mejorar el uso de CSS Grid en interfaces de usuario simples.
